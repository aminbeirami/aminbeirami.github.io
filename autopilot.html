Autopilot System For Leisure Boats Based on Android Platform and NMEA Network.

Description: I developed an autopilot system capable of manually and automatically navigating a leisure boat. The user interface implemented on an Android device and the controller is implemented on an embedded system with ARM Cortex M7 processor. In this project, the android device collects data from boatâ€™s heading sensor and Rudder angle indicator through NMEA 0183 network and sends information to the controller via NMEA 2000 network. The controller receives information, makes decision and triggers the rudder system to fulfill the order.
As part of the project, I first simulated the control strategy in matlab/Simulink, then developed the embedded systems, a hydraulic rudder actuation system and an Android application to simulate the process. Then the developed system was tested and passed successfully on a real leisure boat.
Read my published work in Korean Society of Marine Engineering Journal.
Read my master's thesis on this project.

Technologies and tools that I used: Android Studio, Matlab/Simulink, ARM Cortex M7, NMEA 2000 and 0183, Heading sensor, Hydraulic rudder actuator

Programming Languages that I used: JAVA, C++
