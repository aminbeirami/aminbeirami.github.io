Project 1

Truted Relational Temporal Database Using Blockchain With Optimal Table Snapshots For Materialization.

Description: I developed a blockchain based database based on a relational database management system (RDBMS). The database guarantees authenticity of the stored records using immutable and verifiable digital signatures. The database supports query evaluation over the entire timeline of the transaction log. To support efficient query evaluation, I utilized materialized snapshots at selected timestamps. I evaluated three different algorithm to compute the best timestamps to place multiple snapshots so that the overall query load can be evaluated with minimal computation. Part of the work published in IEEE 2018 Bigdata Congress in San Francisco.
I presented the work in computer science seminar at the University of Ontario Institue of Technology. Here is the full system presentation slides.

Technologies and tools that I used: Postgresql, Python Flask, Machine Learning (K-Means clustering, K-nearest neighbors), Cryptography.

Programming Languages that I used: Python, PL/pgSQL, SQL

Project 3

3D Maritime Simulator with 2D Maps Capable of Simulating MSPs of E-Navigation

Description: Me and one other colleagues developed a 3D maritime simulator using Unity 3D. The simulator simulates a real-world environment (Busan Port), gives E-navigation defined information and feedbacks to each vessel navigating in this area and evaluates the effectiveness of these information for a safer navigation. The 3D environment has the option of simulating multiple sea states and weather. We also developed 3 different 2D maps: scenario editor, ECDIS and Radar based on real navigation maps overlaid on Google map to define a scenario and see the location of vessels on the map. The 3D environment was developed using NASA's SRTM v4.1 digital elevation chart database and the 3D bathymetric terrain implemented based on the data extracted from ECDIS maps.
My role as a developer in this project was to write codes in C# to navigate the vessels in the 3D scene and apply real-world physics to the vessels, develope the 2D maps (ECDIS and RADAR) and implement the 3D terrains and objects in the environment.
Read one of our papers in Korean Society of Marine Engineering

Technologies and tools that I used: Unity 3D, Terraland assets, Sundog assets, NASA's elevation maps, ECDIS maps, Google maps, Global mapper, Google Sketchup, 3D Max.

Programming Languages that I used: C#, JAVASCRIPT
